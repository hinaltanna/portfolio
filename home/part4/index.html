Exploratory Data Analysis

Summary statistics and quick graphical representations using NumPy, pandas, matplotlib, seaborn and plotly

View the Jupyter Notebook on Github

Exploratory Data Analysis is the first thing you do to understand your data: using summary statistics and quick graphical representations you can get an overview of your features, spot patterns and anomalies, come up with hypotheses and a plan for more in-depth investigation. Here, I used NumPy, pandas, and a combination of visualisation libraries (matplotlib, seaborn and plotly) to analyse and explore my own data set of data science job listings, that I had previously scraped from Glassdoor and prepared for analysis (see Data Scientist Jobs Project). The aim of this exploration was to learn more about data scientist roles advertised in the UK. This part of the project was inspired by an exploratory data analysis project published on YouTube by data scientist, Ken Jee.

  • The data were imported as a pandas DataFrame
  • Pandas and NumPy functions were used for quick and simple calculations, and to answer questions like, “Which 5 UK cities have the most Data Scientist vacancies?”
  • Data visualisation was used to more easily appreciate and understand the more complex data, e.g., the distribution of ratings given by employees on different aspect of the hiring organisations
