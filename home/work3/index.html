Feature Engineering

Using regular expression-based text-analysis to create additional features

View this project on Github

Here, I did a regular expression-based text-analysis of data scientist job descriptions using data that I had previously scraped from Glassdoor for my Data Scientist Jobs Project to extract features that could be used for both exploratory analyses and clustering algorithms.

I designed the regular expressions and search strategy that I used to pick out qualifications, degree subjects, skills and experiences mentioned by employers in their vacancy descriptions. The boolean results of the text analysis with each regular expression were inserted as features into a pandas DataFrame containing other data about the job listings.
